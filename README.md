# 📊 Sales Analysis Dashboard (Excel)

An interactive **Sales Analysis Dashboard** built using **Microsoft Excel** to track revenue, orders, customer performance, category contribution, and monthly sales trends. The dashboard provides a clear visual representation of sales data through KPI cards, Pivot Charts, Slicers, and Timeline filters.

## 🎯 Project Objective

The objective of this dashboard is to:

* Monitor overall sales performance.
* Analyze revenue across different regions.
* Compare category-wise sales contribution.
* Understand customer segmentation.
* Track monthly sales trends.
* Enable interactive filtering for quick business insights.

## 📈 Dashboard Visualizations

### 1. Revenue by Region (Column Chart)

| Region  | Revenue    |
| ------- | ---------- |
| Central | ₹41,288.34 |
| East    | ₹59,288.39 |
| North   | ₹50,808.31 |
| South   | ₹36,398.75 |
| West    | ₹41,408.68 |

**Insight:** East region generates the highest revenue among all regions.

### 2. Revenue by Category (Pie Chart)

**Insight:**

* Electronics contributes the largest share of revenue.
* Furniture contributes a moderate share.
* Appliances contribute the smallest share.

### 3. Customer Segment Analysis (Donut Chart)

| Segment  | Share |
| -------- | ----- |
| Premium  | 37%   |
| Standard | 36%   |
| Basic    | 27%   |

**Insight:** Premium customers form the largest customer segment.

### 4. Monthly Sales Trend (Line Chart)

**Insights:**

* Highest sales observed in June (₹16,009.72).
* Strong sales performance in October.
* Lowest sales recorded in November.
* Helps identify seasonal demand patterns.

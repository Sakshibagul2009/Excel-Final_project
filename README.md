📌 Project Description

The Sales Analysis Dashboard is an Excel-based business intelligence project designed to analyze sales performance, customer behavior, and revenue trends. The dashboard provides interactive visualizations and key performance indicators (KPIs) that help businesses make data-driven decisions.

🚀 Features
Interactive Sales Dashboard
Revenue Analysis
Customer Insights
Order Performance Tracking
Average Order Value Analysis
KPI Monitoring
Dynamic Charts and Visualizations

📊 Key Metrics

Total Revenue
Total Orders
Total Customers
Average Order Value (AOV)
Sales Trends
Customer Distribution

💻 Tools Used

Microsoft Excel
Pivot Tables
Pivot Charts
Slicers
Conditional Formatting
Dashboard Design

▶️ How to Use

Open the Excel workbook.
Navigate to the Dashboard sheet.
Use filters and slicers to explore data.
Analyze KPIs and charts for business insights.

🎯 Objectives

Monitor sales performance
Identify revenue trends
Understand customer purchasing behavior
Support business decision-making through data visualization

📈 Insights Generated

Top-performing sales periods
Customer purchasing patterns
Revenue distribution analysis
Order trend evaluation
Business growth indicators
